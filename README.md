# Llyods-Banking-Group--Data-Science-Virtual Internship

## Description
This project focuses on predicting customer churn through analytics and machine learning algorithms thereby discovering ways to enchance customer retention.

## Datasets
The datasets was provided in excel format and in total comprised of 5 datasets: 
- Customer Demographics
- Transaction History
- Customer Service Interactions
- Online Activity
- Churn Status
These datasets provide comprehensive insights into customer behaviour and engagement.

## Installation
Install all necesary libraries to run the script, including scikit-learn, Seaborn, XGBoost, NumPy, MatPlotLib, LightGBM

## Usage
Clone this repository
Ensure you have the dataset file Customer_Churn_Data_Large.xlsx in the project directory
Open the Jupyter notebook: Lloyds_Group_Forage_Project.ipynb
Run the cells sequentially to reproduce the analysis
